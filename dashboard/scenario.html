Custom stress builder

set peak shock per pillar · adjust timing · chart and table update live

Set the peak shock for each pillar (+ = favorable, − = adverse). The shock phases in linearly from 0 at T+0 to full magnitude at the peak month, then follows the selected decay path.

How scenarios work: Named macro scenarios apply structured pillar-level shocks that phase in linearly from 0 to peak magnitude over the scenario timeline, then persist or fade. Historical scenarios (GFC, COVID) use delta-replay — the actual composite path from each episode anchored to today's composite, answering "what would have happened if the MRS started at today's level?" Indicator columns distribute pillar shocks equally across each pillar's indicators. All computation is browser-side using the same pillar weights as the production model.
