Custom stress builder

set peak shock per pillar · adjust timing · chart and table update live

Set the peak shock for each pillar (+ = favorable, − = adverse). The shock phases in linearly from 0 at T+0 to full magnitude at the peak month, then follows the selected decay path.

How the scenario engine works: All paths — baseline, macro scenarios, historical replays, and custom — start from today's 13 indicator z-scores and flow through the production MRS scoring chain: indicator z-score → pillar score (equal-weight mean within pillar) → composite (weighted average by pillar weight) → display score (composite + 3, clamped 1–5). No final score is directly manipulated.

Baseline / Optimistic / Pessimistic: built from real indicator-level forecast paths in forecast_inputs.json — consensus forecasts, market-implied paths, or calibrated estimates for all 13 indicators — rescored through the full MRS engine. These three lines are always visible. Macro scenarios (stress overlay, dotted): each named scenario defines indicator-level z-score shocks that build linearly from 0 to peak magnitude over the specified months, then persist, slow-fade (→50%), or fast-fade (→0) per scenario design. Click a chip to add the overlay; click again to remove it. Historical delta-replay: applies the actual month-by-month indicator z-score changes from each historical episode to today's readings (today_z + hist_delta[t]). After the episode window closes, the path mean-reverts to zero. Custom builder: pillar-level shocks are distributed equally across all indicators within each pillar, then scored normally.

Historical episodes: GFC (Sep 2007–Sep 2008) · COVID (Jan–Sep 2020) · Inflation Shock (Jan–Dec 2022) · Growth Slowdown (Jul 2015–Feb 2016) · Sovereign Crisis (Apr–Oct 2011) · Q4 Equity Rout (Oct 2018–Jun 2019).
